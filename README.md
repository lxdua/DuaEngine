# [DuaEngine] 🎮

一个基于 **C++** 和 **OpenGL** 的玩具游戏引擎，做着玩图一乐。

---

## 🛠️ 技术栈

- **语言**: C++17
- **图形API**: OpenGL
- **UI框架**: ImGui Docking
- **窗口管理**: GLFW
- **数学库**: GLM
- **日志库**：等我感觉麻烦了再补
- **构建工具**: 以后补，应该是CMake

---

## 📥 安装与运行

### 依赖项
- **Visual Studio 2022**
- [GLFW](https://www.glfw.org)
- [GLEW](http://glew.sourceforge.net)
